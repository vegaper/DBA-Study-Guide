1. Configuración inicial --------------

git config --global user.name "Tu Nombre"
git config --global user.email "tu@email.com"

2. Estado del repositorio --------------

git status # Muestra archivos modificados, sin seguimiento, en staging, etc.

3. Añadir cambios --------------

git add archivo.txt
git add . # add . añade todos los cambios.
 
4. Guardar cambios (commit) --------------

git commit -m "Mensaje del commit"

5. Ver historial --------------

git log
git log --oneline --graph --decorate

6. Crear y cambiar de rama --------------

git branch nombre-rama
git checkout nombre-rama
git checkout -b nombre-rama #Crear y cambiar en un solo paso

7. Ver ramas --------------

git branch
git branch -vv

8. Fusionar ramas (merge) --------------
Desde master o la rama destino:
git merge nombre-rama

9. Subir cambios al remoto --------------

git push origin master
git push origin nombre-rama

10. Descargar cambios del remoto --------------

git pull origin master # = fetch + merge

Actualizar referencias sin mezclar:
git fetch origin # Actualiza las referencias del remoto pero NO toca tu rama local.
Después de un fetch puedes inspeccionar: git log origin/master    git diff master origin/master

11. Ver diferencias --------------

git diff
git diff --staged

12. Deshacer cambios --------------

Descartar cambios en un archivo:
git restore archivo.txt

Volver al último commit:
git reset --hard

Deshacer un git add:
git reset archivo.txt

13. Borrar ramas --------------

Local:

git branch -d nombre-rama
git branch -D nombre-rama   # forzar

Remoto:

git push origin --delete nombre-rama

14. Clonar un repositorio --------------

git clone URL

15. Ver remotos --------------

git remote -v
